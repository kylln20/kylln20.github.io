Arrays

Table of Contents Here

Basics

An array is a set block of consecutive memory locations
Each location contains a piece of data called an element
All elements in an array have to be of the same data type

  • For example, you can use an array to store marks, but you cannot store some marks with integer representations (eg. 30) and some marks as decimal representations (eg. 30.0)

Initialization

An array is declared as: dataType[] varName;
The square brackets indicate that it is an array

/**Imagine we have a class representing a cat-only vet clinic
This array represents the registered patients in that clinic*/
private Cat[] patients;

When initializing an array, either length can be set or values can be set

To set length after a declaration:
  arrayName = new dataType[length];
To set length with a declaration:
  dataType[] arrayName = new dataType[length];

When this is done, all elements in the array will be set to the default value for that data type. For example:

  • int array elements are set to 0
  • boolean array elements are set to false
  • object (such as String) array elements are set to NULL
To set values after a declaration:
  arrayName = {element1, element2, element3, ...};
To set values with a declaration:
  dataType[] arrayName = {element1, element2, element3, ...};

/** in the CatClinic class: */
private boolean[] bookedTimeSlot = new boolean[20]; /** setting length */
private String[] inVets = {"Dr. Anders", "Dr. Carmen", "Dr. Derrick", "Dr. Boris"}; /** setting values */

Elements and Indexes

Each element is kept track of based on its location in the block with indexes/indices
Since arrays are allocated a set block of memory space, they also have a set length. This length can be accessed with arrayName.length
There are length number of elements, so there must be a set number of indexes. The indices of an array start at 0 and go up to (array.length)-1.
If an index is used that doesn't exist for an array, one's code won't compile. This error is called an ArrayIndexOutOfBoundsException.

Example: Say that there is an array of pet names
  String[] catNames = {"Jiji", "Haku", "Totoro", "Teto", "Susu"};

Indexes 0 1 2 3 4
Names Jiji Haku Totoro Teto Susu
Indices can be given as constants, as variables, and even as expressions, as long as it evaluates to an int value

/**Using a constant*/
System.out.println(catNames[0]);/**Prints Jiji*/
/**Using a variable*/
int temp = 2; System.out.println(catNames[temp]);/**Prints Totoro*/
/**Using an expression */
System.out.println(catNames[Math.pow(2, 2)]);/**Prints Susu*/
