Basics of OOP

Coding and Reality

Object Oriented Programming: A method of programming based on the concept of “Objects” which are meant to reflect objects, (or things, or nouns) that exist in real life

Objects in real life have

 - a name
 - characteristics
 - behaviors
Objects in code have

 - an identity
 - variables
 - methods
This represents

 - a space in memory
 - properties/data
 - interactions done by or onto the object

Parts of a Class 

 
    /**
    From an assignment given about 
    object oriented programming
    */
    public class Airplane{ /** identity */  
        /** characteristics */
        private int gasLevel;
        private String landingGear;
        private String doorStatus;
        private int passengers;
        
        public Airplane(){
            gasLevel = 100;
            landingGear = "Down";
            doorStatus = "Open";
            passengers = 100;
        }   
        /** behaviours*/
        public void openDoor(){
            doorStatus = "Open";
        }
        public void closeDoor(){
            doorStatus = "Closed";
        }
    }
